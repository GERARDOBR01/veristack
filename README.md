# Veristack
Motor de verificación visual para retail.
Sistema de 3 capas que combina análisis de imagen con 
conocimiento procesado de manuales para evaluar 
evidencia fotográfica de visual merchandising.

## Stack
- Python (photo_analyzer.py) — análisis objetivo de imagen
- Gemini Pro — procesamiento de manuales
- GPT-4o — evaluación visual con contexto

## Capas de conocimiento
- Capa 1: Básicos de presentación visual (permanente)
- Capa 2: Mecánica de campaña activa (caduca por temporada)
- Capa 3: Criterios específicos por sección y tipo de focal
